This enumeration class inherits from OntoumlEnum and automatically sets the value of each enum member to a CamelCase string corresponding to its name. It provides a clear and standardized way to refer to different ontology representation styles.

Variables:

  • ONTOUML_STYLE (str) – Represents the OntoUML style of ontology representation.

  • UFO_STYLE (str) – Represents the UFO style of ontology representation.

An enumeration representing different stereotypes applicable to properties in OntoUML.

This enum classifies various stereotypes that can be assigned to property elements in an OntoUML model. It extends OntoumlEnum.

Members:

BEGIN: Represents a ‘begin’ stereotype, typically used for temporal properties. END: Represents an ‘end’ stereotype, often associated with the termination of temporal properties.
