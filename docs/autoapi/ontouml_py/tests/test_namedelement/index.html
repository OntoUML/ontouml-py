class ontouml_py.tests.test_namedelement.Project(**data)

Bases: ontouml_py.classes.abstract_syntax.abstract_classes.namedelement.NamedElement

A concrete subclass of NamedElement for testing purposes.

Note that this is not the OntoUML Project, but just a class used for testing.

This class inherits from NamedElement and allows the instantiation of NamedElement objects, which is normally an abstract class and cannot be instantiated directly.

An abstract class representing a named element within an OntoUML model, extending the OntoumlElement class.

This class provides functionality for managing named elements, including their preferred name, alternative names, descriptions, editorial notes, as well as lists of creators and contributors.
