An abstract class representing a named element within an OntoUML model, extending the OntoumlElement class.

This class provides functionality for managing named elements, including their preferred name, alternative names, descriptions, editorial notes, as well as lists of creators and contributors.

Variables:

  • names (list[LangString]) – The preferred names of the element, represented as a list of LangStrings.

  • alt_names (list[LangString]) – A list of alternative names for the element, each represented as a LangString object.

  • description (Optional[LangString]) – A LangString object representing the description of the element.

  • editorial_notes (list[LangString]) – A list of LangString objects containing editorial notes associated with the element.

  • creators (list[str]) – A list of URIs represented as strings identifying the creators of the element.

  • contributors (list[str]) – A list of URIs represented as strings identifying the contributors to the element.
