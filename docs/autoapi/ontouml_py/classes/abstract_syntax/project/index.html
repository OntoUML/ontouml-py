Module for the Project class within an OntoUML model.

The Project class extends NamedElement to include project-specific details such as bibliographic citations, keywords, and landing pages, among others, providing a comprehensive representation of a project’s metadata.

A concrete class representing an OntoUML Project, extending the NamedElement class.

Manages project-related elements such as acronyms, bibliographic citations, keywords, landing pages, and more, providing a comprehensive representation of project metadata.

The ‘elements’ attribute is implemented as a read-only property to maintain control over the list of elements and enforce the inverse relationship with OntoumlElement instances. The actual data is stored in the private attribute ‘_elements’, which can be manipulated via add_element and remove_element methods.

Variables:

  • acronyms (list[str]) – List of acronyms associated with the project.

  • bibliographic_citations (list[str]) – Bibliographic citations related to the project.

  • keywords (list[LangString]) – Keywords describing the project.

  • landing_pages (list[str]) – URLs to landing pages of the project.

  • languages (list[str]) – Languages used in the project.

  • namespace (Optional[str]) – Namespace of the project. Optional.

  • sources (list[str]) – Sources of information for the project.

  • access_rights (list[str]) – Information about access rights for the project.

  • ontology_types (list[str]) – Types of ontologies used in the project.

  • themes (list[str]) – Themes associated with the project.

  • license (Optional[str]) – Licensing information of the project. Optional.

  • contexts (list[str]) – Contexts for which the project is designed.

  • designed_for_task (list[str]) – Tasks for which the project is designed.

  • publisher (Optional[str]) – Publisher of the project. Optional.

add_element(element)

Add an OntoumlElement to the project. Ensures that the element is of the correct type and not a Project itself. Also updates the inverse relationship in OntoumlElement and checks for duplicates.

Parameters:

element (OntoumlElement) – The OntoumlElement to be added to the project.

Raises:

TypeError – If the element is not an instance of OntoumlElement.

Return type:

None
