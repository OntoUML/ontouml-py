ontouml_py.classes.abstract_classes.packageable

Module for the Packageable class within an OntoUML model.

This module introduces the Packageable abstract class, a foundational element in the OntoUML model hierarchy. The Packageable class serves as a base for all elements that can be contained within a Package (here called ‘contents’). It includes an abstract method for initialization and a private attribute to track the package a content belongs to. This design ensures that contents in the OntoUML model can be organized and managed within packages, maintaining model coherence and structure.

Abstract class representing elements that can be contained within a package in an OntoUML model.

This class extends ModelElement and includes an additional attribute to link the element to a specific package. The in_package attribute is a private attribute, managed through specific methods, ensuring that each element is correctly associated with a package in the OntoUML model.
