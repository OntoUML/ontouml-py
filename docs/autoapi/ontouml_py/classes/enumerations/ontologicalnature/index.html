An enumeration representing different kinds of ontological natures in OntoUML.

This enum classifies various ontological natures that can be assigned to elements in an OntoUML model. It extends OntoumlEnum.

Members:

ABSTRACT_NATURE: Represents an abstract nature. COLLECTIVE_NATURE: Represents a collective nature. EVENT_NATURE: Represents an event nature. EXTRINSIC_MODE_NATURE: Represents an extrinsic mode nature. FUNCTIONAL_COMPLEX_NATURE: Represents a functional complex nature. INTRINSIC_MODE_NATURE: Represents an intrinsic mode nature. QUALITY_NATURE: Represents a quality nature. QUANTITY_NATURE: Represents a quantity nature. RELATOR_NATURE: Represents a relator nature. SITUATION_NATURE: Represents a situation nature. TYPE_NATURE: Represents a type nature.
