ontouml_py.utils.utils

This utility module contains a collection of helper functions that provide common functionality used throughout the application. These functions are designed to be generic and reusable across various parts of the codebase.

ontouml_py.utils.utils.is_valid_list(item, inner_type)

Recursively checks if all elements in a list are of the specified inner_type.

Parameters:

  • item – The item to be checked, can be a list or a single element.

  • inner_type – The type that items in the list are expected to be.

Returns:

True if all elements are of the inner_type, False otherwise.

Return type:

bool

ontouml_py.utils.utils.validate_and_set(instance, attr_name, att_value, attr_type, optional=True)

Securely sets the attribute value of an instance after type checking.

Parameters:

  • instance (Any) – The instance of the object on which the attribute will be set.

  • attr_name (str) – The name of the attribute to set.

  • att_value (Any) – The value to be assigned to the attribute.

  • attr_type (Union[Type[Any], List[Type[Any]]]) – The type the attribute should be. Can be a type or a list of types.

  • optional (bool) – Specifies if the attribute is optional (can be None).

Raises:

TypeError – If the value is not of the expected type.

Return type:

None
