ontouml-py.constants

The constants sub-package provides essential constants that facilitate the use of the OntoUML vocabulary in Python code.

These constants align with the OntoUML vocabulary version 1.1.0, available at https://w3id.org/ontouml/vocabulary/v1.1.0. For more information about the OntoUML vocabulary, refer to the official documentation at: https://w3id.org/ontouml/vocabulary
