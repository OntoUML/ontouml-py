ontoumlpy.functions.func_mappings

This module handles the mapping between OntoUML terms and OUElement objects.

It provides functions to retrieve OUElement objects from OntoUML terms and vice versa.

ontoumlpy.functions.func_mappings.ou_get_term_from_element(ouelement)

Return the corresponding OUTerm object for a given OUElement object or its class name as a string.

This function iterates through the mapping dictionary to find the corresponding OUTerm object. Every OUElement object is assumed to be mapped to an OUTerm object.

Parameters:

ouelement (Union[_OUElement, str]) – The OUElement object or its class name as a string.

Raises:

OUUnmappedOUElement – If the OUElement object is not found in the mapping.

Returns:

The corresponding OUTerm object.

Return type:

URIRef

Return the corresponding OUElement object for a given OUTerm or URIRef/str object.
