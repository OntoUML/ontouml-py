ontoumlpy.functions.func_class.ou_get_direct_superclasses(ontouml_graph, ou_class_id)

Retrieve a list of IDs (as URIRefs) of the direct superclasses of a specified OntoUML class from a given OntoUML RDF graph.

Allows the user to specify a list of restricted types. Superclasses of those types will not be included in the returned list.

Parameters:

  • ontouml_graph (rdflib.Graph) – The RDF graph containing OntoUML model data.

  • ou_class_id (str) – The URI of the OntoUML class for which to retrieve superclasses.

Returns:

A list of URIRefs representing the direct superclasses of the specified OntoUML class. The list may be empty.

Return type:

list[URIRef]
