ontoumlpy.classes.ouenumeration._ouenumeration

This module provides the _OUEnumeration class, an enumeration base class which provides a method to retrieve all members of an enumeration. Enumerations inheriting from _OUEnumeration can leverage this method to fetch all of their members, simplifying tasks like iteration over all values.
