ontoumlpy.classes.ouenumeration.enumerations_other

This module defines various OntoUML Stereotypes related to relations, properties, aggregation kinds, and ontological natures used in OntoUML modeling.

Each enumeration represents a specific grouping of stereotypes, providing a convenient and semantically meaningful way to access and utilize the stereotypes in OntoUML modeling.
