Bases: ontoumlpy.classes.ouelement._ouelement._OUElement

Class representing an OntoUML Class element with facilities to manage and access class properties.

OUClass, derived from _OUElement, further specializes the base class to facilitate representation and manipulation of OntoUML Class elements. It does not verify the existence or validity of the provided URI references within an OntoUML graph, delegating such checks to potential subclasses or external utility functions.
