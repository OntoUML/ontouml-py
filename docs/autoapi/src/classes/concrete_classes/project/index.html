A concrete class representing an OntoUML Project, extending the NamedElement class.

Manages project-related elements such as acronyms, bibliographic citations, keywords, landing pages, and more, providing a comprehensive representation of project metadata.

The ‘elements’ attribute is implemented as a read-only property to maintain control over the list of elements and enforce the inverse relationship with OntoumlElement instances. The actual data is stored in the private attribute ‘_elements’, which can be manipulated via add_element and remove_element methods.

Variables:

  • acronyms (set[str]) – List of acronyms associated with the project.

  • bibliographic_citations (set[str]) – Bibliographic citations related to the project.

  • keywords (set[str]) – Keywords describing the project.

  • landing_pages (set[str]) – URLs to landing pages of the project.

  • languages (set[str]) – Languages used in the project.

  • namespace (Optional[str]) – Namespace of the project. Optional.

  • sources (set[str]) – Sources of information for the project.

  • access_rights (set[str]) – Information about access rights for the project.

  • ontology_types (set[str]) – Types of ontologies used in the project.

  • themes (set[str]) – Themes associated with the project.

  • license (Optional[str]) – Licensing information of the project. Optional.

  • contexts (set[str]) – Contexts for which the project is designed.

  • designed_for_task (set[str]) – Tasks for which the project is designed.

  • publisher (Optional[str]) – Publisher of the project. Optional.

  • root_package (Optional[Package]) – Root package of the project. Optional.

  • representation_style (OntologyRepresentationStyle) – Style of ontology representation used in the project.
