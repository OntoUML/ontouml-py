.. py:module:: ontouml_py.classes.abstract_classes.modelelement

.. autoapi-nested-parse::

   This module defines the ModelElement class, which represents a model element in an OntoUML model. It inherits properties from both NamedElement and ProjectElement, and includes additional features specific to model elements.
